- Open Terminal
- `ssh-keygen`
- Generating public/private rsa key pair.
  - Enter file in which to save the key (/Users/kaushik/.ssh/id_rsa): kaushik
  - Enter passphrase (empty for no passphrase):
  - Enter same passphrase again:
  - Your identification has been saved in kaushik
  - Your public key has been saved in kaushik.pub
- Your fingerprint and SHA256 is generated along with two files one of which has the public key and the other contains the private key.
